Capabilities

Everything you need to
code from chat

Forge Code brings a full development workflow into Telegram — from asking questions to shipping patches.

⚡

AI Chat with Streaming

Real-time SSE streaming from GPT-4o or Claude. Your full project context is automatically included so the AI understands your codebase.

🔀

Reviewable Patches

AI responses are parsed into structured diffs. Review file-by-file, approve or reject, then apply — versioned updates with full history.

👥

Team Workspaces

Multi-workspace support with owner, editor, and viewer roles. Invite collaborators by Telegram username, share projects seamlessly.

📁

File Management

Full file tree with CodeMirror editor, syntax highlighting, multipart upload, and automatic version tracking on every change.

🤖

Telegram Bot

Ask coding questions directly in any Telegram chat. Supports /ask, /reset, and group conversations with mention detection.

🔒

Secure by Default

Telegram HMAC-SHA256 auth, JWT sessions, rate limiting, Zod validation, and workspace-level isolation for all data.

Workflow

From question to shipped code

Four steps. No context switching. All inside Telegram.

01

Ask

Describe what you need in natural language. Your project files are loaded as context automatically.

02

Review

AI generates structured patches with unified diffs. See exactly what changes — file by file, line by line.

03

Approve

Accept or reject each patch. Discussion continues until the code is right. Full audit trail preserved.

04

Apply

One tap to apply changes. Files are updated with version history. Your project moves forward.
